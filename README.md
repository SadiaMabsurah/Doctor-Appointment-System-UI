<h2>Feature List</h2>

<p><b>1. User Registration</b> – New patients and doctors can create an account in the system.</p>

<p><b>2. User Login, Logout, and Password Reset</b> – Secure access and password recovery system.</p>

<p><b>3. Patient Profile Management</b> – Patients can update personal and medical details.</p>

<p><b>4. Doctor Profile Management</b> – Doctors can manage their profile, qualifications, and specialization.</p>

<p><b>5. Doctor Search</b> – Patients can search doctors by name, specialty, or availability.</p>

<p><b>6. View Doctor Information</b> – Patients can view doctor experience, timing, and details.</p>

<p><b>7. Appointment Booking</b> – Patients can book appointments with doctors.</p>

<p><b>8. Appointment Rescheduling</b> – Patients can reschedule existing appointments.</p>

<p><b>9. Appointment Cancellation</b> – Patients can cancel appointments anytime.</p>

<p><b>10. Appointment Status Tracking</b> – Track appointment status (pending, confirmed, completed).</p>

<p><b>11. Doctor Schedule Management</b> – Doctors can set and update working hours.</p>

<p><b>12. Time Slot Management</b> – System divides availability into slots for booking.</p>

<p><b>13. Appointment Approval/Rejection</b> – Doctors can approve or reject appointments.</p>

<p><b>14. Patient Record Management</b> – Doctors can maintain visit history and notes.</p>

<p><b>15. Prescription Management</b> – Doctors can create digital prescriptions.</p>

<p><b>16. Admin Dashboard</b> – Admin can monitor system activities.</p>

<p><b>17. Doctor Management</b> – Admin can add, edit, delete doctor accounts.</p>

<p><b>18. Patient Management</b> – Admin can manage patient accounts.</p>

<p><b>19. Appointment Management</b> – Admin can view and control all appointments.</p>

<p><b>20. Department/Specialty Management</b> – Manage medical departments like cardiology, ENT, etc.</p>

<p><b>21. Reports and Statistics</b> – View system analytics and summaries.</p>

<p><b>22. Role-Based Access Control</b> – Access restricted based on user roles.</p>

<p><b>23. Secure Authentication</b> – Protected login and validation system.</p>

<p><b>24. Email/SMS Confirmation</b> – Notifications for booking, canceling, rescheduling.</p>

<p><b>25. Feedback and Rating System</b> – Patients can rate doctors after visits.</p>

<p><b>26. Doctor Booking Limit Management</b> – Limit appointments per doctor per day.</p>

<p><b>27. Prescription Email Delivery</b> – Send prescriptions via email.</p>

<p><b>28. Search Doctors by Consultation Fee</b> – Filter doctors by budget.</p>

<p><b>29. Feedback and Rating System </b> – Detailed reviews after appointments.</p>
